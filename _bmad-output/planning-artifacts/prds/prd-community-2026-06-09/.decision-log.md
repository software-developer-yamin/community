## 2026-06-09 — OQ 6 Resolution + Close

**Decision:** LiveKit Cloud. (AI-recommended, user-confirmed.)
**Rationale:** Global mesh SFU — BD user on cell roaming to IN edge still hits nearest edge node, which a self-hosted single-server-per-room topology cannot match. FR-6 reconnect SLOs (95% on 1s blip, 80% on 5s blip) and SM-1 (disconnect ≤5%) are achievable on Cloud and not on self-host. Cloud also pre-paves Open Question 5 (AI agent v2 — native to Cloud Agents framework). Migration path stays open because the LiveKit SDK is portable; the connection endpoint is the only delta.
**Implications folded into PRD:**
- `prd.md` frontmatter `status: draft` → `status: final`. Future `bmad-prd` invocations distinguish this from in-progress drafts.
- §8 OQ 6 marked RESOLVED; owner changed to "Engineering" (sandbox creds + production project setup).
