### 2026-06-12 — Feature Selection
- **Decision**: Build a Content Recommendation Engine for the language learning community.
- **Rationale**: The app already collects CEFR levels, pronunciation scores, and profile embeddings. A recommendation engine completes the learning loop (assess → recommend → practice → reassess) and leverages existing data/infrastructure without new external dependencies.
- **Alternatives considered**: Social feed, gamification, course marketplace. Recommendation engine ranked higher because it directly uses existing data and has clear user value.
